About the Project

Scientific context

Multi-environment trials are used to evaluate genotype performance under contrasting environmental conditions.

Genotype-by-environment interaction occurs when genotypes respond differently to environmental variation. Its analysis supports the identification of productive, adapted, responsive, and stable genotypes.

The Additive Main Effects and Multiplicative Interaction model represents interaction through a bilinear decomposition based on interaction principal components. Tree-based learning algorithms provide a complementary framework for representing non-linear response structures.

Shapley Additive Explanations are used to decompose Random Forest predictions and quantify the contribution of genotype and environmental information to individual predictions.

Project objective

The objective of this project is to provide a reproducible analytical workflow for investigating genotype-by-environment interaction through the joint use of:

  • Random Forest modeling;
  • leave-one-environment-out validation;
  • SHAP decomposition;
  • AMMI analysis;
  • AMMI Stability Value;
  • SHAP effect and stability measures;
  • classical Simultaneous Selection Index;
  • SHAP-based Simultaneous Selection Index;
  • genotype-rank dynamics;
  • stability concordance;
  • Spearman rank correlation.

The analytical sequence follows the methods and results considered in the associated manuscript.

Data set

The maize data set contains grain-yield observations from:

  • nine genotypes;
  • twenty environments;
  • four replicates per genotype-by-environment combination.

The complete experiment contains 720 observations.

Variable Description
ENV Environment identifier
GEN Genotype identifier
REP Replicate identifier
GY Grain yield in kilograms per hectare

The replicate-level observations are summarized by genotype and environment before the predictive and interaction analyses.

Analytical organization

The complete pipeline is presented in a single sequential analysis page containing:

  1. package loading;
  2. data import and preparation;
  3. phenotypic visualization;
  4. environmental-index calculation;
  5. Random Forest validation;
  6. final Random Forest modeling;
  7. DALEX model explanation;
  8. SHAP contribution calculation;
  9. AMMI interaction decomposition;
  10. AMMI and SHAP stability measures;
  11. classical and SHAP-based simultaneous selection indices;
  12. genotype rankings across six selection criteria;
  13. rank-dynamics visualization;
  14. stability-concordance analysis;
  15. Spearman rank-correlation analysis.

All executable analytical code is displayed directly in the R Markdown document. Computationally intensive results may be loaded from versioned output objects after their original calculation has been documented and validated.

Reproducibility framework

The project uses workflowr to integrate:

  • literate programming;
  • Git version control;
  • computational documentation;
  • reproducibility checks;
  • versioned figures and results;
  • website publication.

Each published page is associated with a specific Git commit.
